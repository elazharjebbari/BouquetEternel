Handmade is an elegant HTML Template specially developed and designed for setting up a website for Handmade Jewellery, Handmade Gifts, Handmade Cards, Embroidery works and various Artworks. Handmade is developed on the latest industry standards and trends, Bootstrap HTML5 and CSS3. The most important sections of an art shopping website are also added in this HTML Template.

This visually stunning Retina Ready HTML Template has beautiful Google Fonts, Font Awesome and Flaticons embedded to make your products look even more good. You will have many design possibilities as this template has 4 different variations of Home Pages. The compatibility of this Handmade HTML Template with all the web browsers out there is being properly tested. This Template follows all the latest coding standards and has clean and easy to understand codes. Handmade HTML Template is a bliss for the artists who wants to sell their fine work of art online because of its finest features and splendid design.
